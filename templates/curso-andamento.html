Introdução à Programação Web

Aprenda os conceitos básicos e comece a construir seu próprio site

Neste curso de Introdução à Programação Web, você aprenderá os fundamentos da criação de sites e páginas web, passando pelas tecnologias essenciais para o desenvolvimento moderno. O curso está estruturado para iniciantes e oferece uma base sólida sobre como a web funciona, como criar páginas estáticas com HTML, estilizar com CSS e tornar essas páginas interativas com JavaScript. Além disso, você aprenderá a usar o Bootstrap, um framework popular para criar layouts responsivos e modernos. Ao longo do curso, você será guiado passo a passo, começando pela estrutura básica de uma página web e avançando para a construção de páginas mais complexas e interativas. Ao final, você terá os conhecimentos necessários para criar sites próprios, entender o funcionamento da web e dar os primeiros passos no desenvolvimento de aplicações web.
